## List of contents
| **File name** | **Description** |
| :---- | :---- |
| **EMInfo.sh** | The main program wrapped in a shell script to run EMReady2 |
| **environment.yml** | Required packages for Python virtual environment of EMReady2 |
| **interp3d.f90** | Fortran source code for the interpolation of EM grid |
| **eminfo.py** | Python script of EMReady2 |
| **model.py** | Pytorch implementation of 3D Swin-Conv-UNet |
| **utils.py** | Python utilities |
